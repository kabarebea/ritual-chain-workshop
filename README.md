# OracleAIBounty – Decentralized Oracle Verification

This contract uses **multiple oracles** to provide independent judgments. Winners are determined by majority consensus among oracles, with staking and slashing for accountability.

## How it works
1. **Commit phase**: Participants submit hashed answers
2. **Reveal phase**: Participants reveal their answers
3. **Oracle phase**: Oracles stake RIT and vote for winners
4. **Finalization**: Winner = majority vote among oracles
5. **Slashing**: Oracles that vote against the winner lose 50% of stake

## Why oracles?
Decentralized judgment with multiple independent validators.
